# Contributing

Keep the project focused on the local SQLite tutor loop.

Good contributions:

- clearer setup
- better mock mode
- small UI improvements
- provider adapter cleanup
- local persistence fixes
- documentation fixes

Out of scope:

- managed production database support
- paid-account systems
- production auth
- large platform orchestration
- private adaptive planning systems
- graph-based curriculum systems

Run the local type check before submitting changes:

```powershell
npm run check
```
